# Docker

## Construir imágenes

```bash
docker compose build
```

## Levantar servicios

```bash
docker compose up
```

## Detener servicios

```bash
docker compose down
```

## Reconstruir y levantar

```bash
docker compose up --build
```

## Ver logs

```bash
docker compose logs
```

## Eliminar contenedores y volúmenes

```bash
docker compose down -v
```

## URLs

- Frontend: http://localhost:3000
- Backend: http://localhost:8080
